# SQL Sales Analytics Dashboard

## Overview

This project demonstrates an end-to-end sales analytics workflow using SQL and Power BI.

The goal of this project is to simulate a real business scenario where a company needs to analyze its sales performance across products, retailers, and regions in order to support data-driven decision making.

The project includes:

- relational database design
- synthetic data generation
- analytical SQL queries
- an interactive Power BI dashboard

The final result is a business intelligence dashboard that helps stakeholders quickly understand revenue performance and identify growth opportunities.

## Business Questions

This project answers several key business questions:

1. What is the total revenue generated by the company?
2. How does revenue change over time?
3. Which product categories generate the most revenue?
4. Which regions contribute the most to sales?
5. What are the top 10 products by revenue?
6. Which retailers generate the highest revenue?

These insights help identify the most profitable products, regions, and business partners.
